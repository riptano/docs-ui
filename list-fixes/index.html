Get Started

Dive straight into Astra Vector with this concise guide tailored for developers. Learn the essentials of how to connect to a database, ingest vectors, and perform a similarity search.

Quickstart Create Astra Account

from astrapy.db import AstraDB

# Connect to a database
db = AstraDB(token=TOKEN, api_endpoint=URL)

# Ingest vectors into your collection
col = db.collection(collection_name="test")
col.insert_many(documents=DOCUMENTS)

# Find the closest vectors
col.vector_find([0.15, 0.1, 0.1, 0.35, 0.55])
